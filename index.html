RHYTHM Pyro Color Wheel

Pick a harmony mode, drag the main handle around the wheel, and use the generated palette to plan cleaner, more intentional color moments in a pyromusical.

How to use it: drag the white-ring handle to set your anchor color. The related harmony colors move automatically. Monochromatic mode keeps the same hue and varies value for easier scene planning.

Pyro tip

Use complementary and triadic scenes when you want strong contrast and obvious visual punctuation.

Pyro tip

Use analogous or monochromatic scenes when you want smoother emotional movement and less visual tension.

Generated Palette

Tip for pyromusicals: white, silver, or gold often work as separation colors even when they are not part of a strict harmony. Use them to break up dense same-hue passages.
